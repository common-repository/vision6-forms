Grow your audience with WordPress + Vision6 forms.

== Description ==

Simply download the Vision6 plugin and implement your existing forms on your website using a shortcode. Capture website visitor details through newsletter subscriptions, enquiry forms, surveys or feedback forms and watch your lists grow.

The WordPress + Vision6 plugin features include:

* Easy download and setup
* Publish Vision6 forms on any page of your WordPress site
* Use WordPress Widgets to manage your forms
* New contact information synced in real time within your Vision6 account

**If you don’t have a Vision6 account yet, you can [get started today](https://www.vision6.com.au) for free.**

== Installation ==

#### Using the WordPress Dashboard

1. Navigate to the ‘Add New’ in the plugins dashboard
2. Search for `vision6-forms`
3. Click the ‘Install Now’ button
4. Click the 'Activate' button, once it is available

#### Uploading via the WordPress Dashboard

1. Navigate to the ‘Add New’ in the plugins dashboard
2. Navigate to the ‘Upload’ area
3. Select `vision6-forms.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

#### Using FTP

1. Download `vision6-forms.zip`
2. Extract the `vision6-forms` directory to your computer
3. Upload the `vision6-forms` directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==

= Can I design my own forms? =

Yes! Designing your own forms is easy, all you need to do is:

* Login to [Vision6](https://www.vision6.com.au)
* Click “Lists and Forms” in the left sidebar
* Double click an existing list or create a new one
* Click on the “Edit Forms” icon or create a new form

= Where will the contact details be sent after a user submits a form? =

Your form details will be available in Vision6 as a contact.

= Can forms be created within WordPress? =

No, you will need to login to your Vision6 account to edit and create your forms.
